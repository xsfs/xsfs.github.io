常用Terminal命令

Aug 31, 2019点击

ls：显示目录文件

cd ～ 进入用户根目录；
cd / 进入根目录；
cd .. 返回上级目录；

pwd：显示当前路径

mkdir dirname 创建一个目录
rmdir dirname 删除一个目录

cat主要有三大功能：

  1. 一次显示整个文件。$ cat filename
  2. 从键盘创建一个文件。$ cat > filename，只能创建新文件,不能编辑已有文件。
  3. 将几个文件合并为一个文件： $cat file1 file2 > file

git 几个常用命令：

克隆项目：git clone git@xxxxx.git

  1. git add .
  2. git commit －m “” 1和2 类似于 git commit －a
  3. git pull 拉取
  4. git push 提交
  5. git status 查状态
  6. git tag的命令：eg.
  7. 查询当前分支版本号 git rev-parse HEAD 短的用 git rev-parse –short HEAD
  8. 本地恢复到某个版本 git reset –hard 版本号 　　

新建分支：git branch dev
切换分支：git checkout dev

vi的命令：

  1. 在默认的”指令模式”下按 i 进入编辑模式
  2. 在非指令模式下按 ESC 返回指令模式
  3. 在”指令模式”下输入:
    :w 保存当前文件
    :q 退出编辑,如果文件为保存需要用强制模式
    :q! 强制退出不保存修改
    :wq 组合指令, 保存并退出
  4. 在”指令模式”下移动:
    h 左
    j 下
    k 上
    l 右

关机命令：

  1. 立即关机sudo halt或者sudo shutdown -h now
  2. 10分钟后关机sudo shutdown -h +10
  3. 晚上8点关机sudo shutdown -h 20:00
  4. 立即重启sudo reboot 或者sudo shutdown -r now

推荐工具：iTerm2
control + a: 到行首
control + e: 到行末
control + U: 删除到行的开头
control + K: 删除到行的结尾

命令回放: Cmd+Alt+B
清屏：cmd+R
